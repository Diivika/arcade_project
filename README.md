**Это вертикальный платформер с восхождением вверх. Ваша цель забраться до финиша, избегая препятствий, сражаясь с
врагами.**

### Цели проекта:

* Реализовать рабочего персонажа
* Анимация главного персонажа
* Сделать уровень
* Стартовое окно
* Некий подсчет результатов
